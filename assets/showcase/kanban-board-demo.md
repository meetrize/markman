# 看板待办示例

用分栏模拟看板三列：**待办**、**进行中**、**已完成**。每列用表格列出任务，适合 sprint 站会、个人待办或小组任务一览。

--- column width=60%
### 说明

- 每列一张表，列宽在分栏内上下对齐
- 可继续加行：在对应列的表格末尾新增一行即可
- 宽屏三列并排；窄屏会自动纵向堆叠

:::
